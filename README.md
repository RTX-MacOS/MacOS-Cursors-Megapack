# MacOS Cursors Megapack By c9pz
# How to install
1: Choose your choice.
2: Choose the size.
3: Find the file named install.inf, Right-click on it and click "Install".
4: It will automatically open the mouse settings, Choose the macOS scheme, And done.
5: Enjoy!
